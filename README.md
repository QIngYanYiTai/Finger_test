# Dilimb_Finger_test
V1 is Based on ESP32-C3 and arduino nano as a simple test.
V2 has rebuild the whole structure. And the original casing is no longer suitable.
For the slave module, added magnet encoder support (MT6826S), changed  the motor driver to TMI8180A (which allow the MCU to monitor the real-time current).
For the master module, used M5stack StickS3, Now support for managing multiple slave devices and viewing their status in real time is now available.
